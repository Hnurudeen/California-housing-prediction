# California Housing Price Prediction

Regression model that predicts median house values across California census block groups using the 1990 US Census data (Pace & Barry, 1997).

|---|---|

| MAE | **$29,693** |
| RMSE | **$43,935** |

Top predictive feature: `median_income` (26.9% importance), followed by geographic coordinates (`latitude`, `longitude`).

## Quickstart

```bash
# Install dependencies
pip install -r requirements.txt

# Run the full pipeline
python run_analysis.py
# or
make run
```

The script will:
1. Download the dataset automatically if not present locally
2. Clean, engineer features, and split 60% train / 20% validation / 20% test
3. Train a baseline Random Forest, then tune with RandomizedSearchCV
4. Select the best model on the validation set, evaluate once on the test set
5. Save the model to `models/` and all figures + report to `reports/`

## Pipeline Design

- **No data leakage** вЂ” split happens before any encoding or scaling; encoders fit on train only
- **Three-way split** вЂ” validation set used for model selection; test set touched exactly once
- **Efficient tuning** вЂ” RandomizedSearchCV (20 iterations Г— 3-fold CV = 60 fits), completes in ~2 min
- **Reproducible** вЂ” fixed `random_state=42` throughout; deterministic pipeline
- **Centralised config** вЂ” all paths and constants in `california_housing/config.py`

## Data Source

1990 California Census, compiled by Pace & Barry (1997).  
Available on [Kaggle](https://www.kaggle.com/datasets/camnugent/california-housing-prices).
